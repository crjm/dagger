Queries

container

Description

Load a container from ID. Null ID returns an empty container (scratch). Optional platform argument initializes new containers to execute and publish as that platform. Platform defaults to that of the builder's host.

Example
query {
    container(id: "alpine") {
        id
    }
}
Response

Returns a Container!

Arguments
Name Description
id - ContainerID
platform - Platform

      ...ContainerFragment
    }
    withSecretVariable {
      ...ContainerFragment
    }
    withoutEnvVariable {
      ...ContainerFragment

    entrypoint

      ...ContainerFragment

    withDefaultArgs {
      ...ContainerFragment
    }
